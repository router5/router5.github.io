Get started

router5 is available in all major formats: ES6, AMD, CommonJS and globals.

It can be installed using npm or bower. Alternatively, you can download a specific version from github. Whether you install router5 from npm, bower or github, you will have access to the same files.

Installation

Bower

bower install router5

npm

npm install router5

Include router5 in your application

Node.js (CommonJS)

Files are available in /dist/commonjs/.

var r5 = require('router5');

var Router5   = r5.Router5;
var RouteNode = r5.RouteNode;

Browser (globals)

router5.js and router5.min.js are available in /dist/browser/. All router5 dependencies (route-node and path-parser) are packaged together, and Router5 and Router5Node are made globally available.
