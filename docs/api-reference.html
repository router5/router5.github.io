Params

opts Object={} The router options: useHash, defaultRoute and defaultParams can be specified.

Methods

router5.setOption(opt, val) Set an option value
Params
opt String The option to set
val * The option value
Return Router5 The Router5 instance
router5.add(routes) Add route(s)
Params
routes RouteNode[]Object[]RouteNodeObject Route(s) to add
Return Router5 The Router5 instance
router5.addNode(name, path, canActivate) Add a route to the router.
Params
name String The route name
path String The route path
canActivate Function A function to determine if the route can be activated. It will be invoked during a transition with `toState` and `fromState` parameters.
Return Router5 The Router5 instance
router5.start(done) Start the router
Params
done Function An optional callback which will be called when starting is done
Return Router5 The router instance
router5.stop() Stop the router
Return Router5 The router instance
router5.getState() Return the current state object
Return Object The current state
router5.isActive(name, params, equality) Whether or not the given route name with specified params is active.
Params
name String The route name
params Object={} The route parameters
equality Boolean=false If set to false (default), isActive will return true if the provided route name and params are descendants of the active state.
Return Boolean Whether nor not the route is active
router5.areStatesDescendants(parentState, childState) Whether two states are descendants
Params
parentState Object The parent state
childState Object The child state
Return Boolean Whether the two provided states are related

router5.buildUrl(route, params) Generates an URL from a route name and route params. The generated URL will be prefixed by hash if useHash is set to true
Params
route String The route name
params Object The route params (key-value pairs)
Return String The built URL
router5.buildPath(route, params) Build a path from a route name and route params The generated URL will be prefixed by hash if useHash is set to true
Params
route String The route name
params Object The route params (key-value pairs)
Return String The built Path
router5.matchPath(path) Match a path against the route tree.
Params
path String The path / URL to match
Return Object The matched state object (null if no match)
router5.navigate(name, params, opts, done) Navigate to a specific route
Params
name String The route name
params Object={} The route params
opts Object={} The route options (replace, reload)
done Function An optional callback (err, res) to call when transition has been performed either successfully or unsuccessfully.
Return Function A cancellation function
