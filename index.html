router5 // Functional, flexible and powerful routing solution

Download (v4.2.1) Install with npm or bower

principles // just a router

  • Implementation and framework agnostic
  • No inversion of control
  • Designed for component trees
  • Route updates are data updates

features // a toolbox for your app

  • Nested routes with high performance matching
  • Allow or prevent activation and deactivation
  • Extensible with middleware and plugins
  • Universal JavaScript support
